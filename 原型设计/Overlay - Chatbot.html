For deposit refunds, the basic process is:

  1. Complete the move-out property inspection
  2. Clean the property and return keys
  3. Landlord refunds within 14 business days

What if the landlord refuses to return my deposit?

If your landlord refuses to return your deposit, follow these steps:

1. Send a written notice first
2. Collect evidence
3. Apply for mediation through this platform
4. Submit a formal claim if needed
