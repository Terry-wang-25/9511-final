Accessibility and Screen Readers

×

Consumer Rights is committed to an accessible experience for all users. Below is an overview of our accessibility features.

Screen Reader Compatibility

This website supports the following screen readers:

  • NVDA — Free screen reader for Windows
  • JAWS — Professional screen reader for Windows
  • VoiceOver — Built-in on macOS / iOS
  • TalkBack — Built-in on Android

Keyboard Navigation

TabMove to the next interactive element
Shift+TabMove to the previous interactive element
EnterActivate a link or button
EscClose dialogs or menus

Other Accessibility Settings

  • 📝 Font Size Adjustment
  • 🎨 High Contrast Mode
  • 📏 Reading Ruler
  • 🌐 Language Switch
Feedback: If you encounter any issue while using assistive technology, please contact us and we will fix it as soon as possible.
